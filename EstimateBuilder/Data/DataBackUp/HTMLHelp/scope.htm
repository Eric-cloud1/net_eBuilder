Scope

 

To setup a program that will hold a table of master soft coded business formulas, that once setup can be used repeatedly by making it a master template and zeroing out the values entered into the takeoff table, but leaving the formulas and their links in place. This would allow users, to have a master Template for each type of construction (single family home, office building, etc.)

 

This application will also have the ability to provide a number of reports for different employees or user, but still using the same data:

1. Summary by Divisions, Sub-Divisions, By Contractor, Product

2. Reports can be tailored for Users in employees

3. Estimator, Owner, Lender, Job supervisors, purchasing agent

4. Exporting of data to other applications
